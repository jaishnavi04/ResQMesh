# 🚨 ResQMesh – Decentralized Emergency Mesh Network

## 📌 Overview
ResQMesh is a decentralized emergency response system that ensures communication even when internet infrastructure fails.

It converts smartphones into mesh nodes, enabling real-time alert propagation using device-to-device communication.

---

## ⚡ Key Features
- 🚨 SOS Emergency Trigger
- 🔗 Multi-hop Mesh Communication
- 📡 Works Without Internet
- 📊 Real-time Alert Propagation
- 🖥️ Live Admin Dashboard
- 📍 Location Tracking
- ⚠️ Priority Classification (High/Critical)

---

## 🧠 How It Works
SOS Trigger → Emergency Packet → Mesh Network → Multi-hop Propagation → Gateway Node → Backend Server → Admin Dashboard → Response Action

---

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Real-time: Socket.io
- Database: MongoDB
- Simulation: JavaScript
- APIs: Google Maps API, Gemini API
